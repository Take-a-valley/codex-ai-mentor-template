# Glossary

| 用語 | 意味 | 備考 |
|---|---|---|
| プロジェクト | タスク、Wiki、WBS、チャットを束ねる単位 | システムの中心概念 |
| タスク | 作業単位 | カンバンで管理する |
| カンバン | タスクをステータスごとに列で表示する方式 | 未着手、進行中、レビュー中、完了など |
| Wiki | プロジェクト内のドキュメント管理機能 | 要件、設計、議事録などを管理 |
| WBS | Work Breakdown Structure | 作業分解構成 |
| MVP | Minimum Viable Product | 最初に作る最小構成 |
