## 技術選定時に確認すること

- なぜその技術を使うのか
- 学習目的に合っているか
- 実務での採用例があるか
- 初学者にとって難しすぎないか
- 長期運用・保守に向いているか
- 代替技術と比べたメリット・デメリットは何か
- プロジェクトの要件に合っているか
