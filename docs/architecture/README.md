# docs/architecture

このディレクトリには、ユーザーが作成する設計書・仕様書を配置します。

## 重要ルール

- このディレクトリ配下のMarkdownは、ユーザー作成の学習成果物として扱います。
- AIはこのディレクトリ配下のファイルを直接編集しません。
- AIはレビュー、助言、雛形案の提示、改善提案のみ行います。

## 想定する設計書

- `requirements.md`：要件定義書
- `basic_design.md`：基本設計書
- `detail_design.md`：詳細設計書
- `db_design.md`：DB設計書
- `api_design.md`：API設計書
- `screen_design.md`：画面設計書
- `test_design.md`：テスト設計書
