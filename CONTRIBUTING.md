# Contributing

このプロジェクトへのコントリビュートを歓迎します。

## 方針

このリポジトリは、Codex等のAIエージェントを「実装代行」ではなく「学習メンター」として使うためのテンプレートです。

PRやIssueでは以下の観点を重視します。

- 学習者が理解しやすいか
- AIに丸投げしない運用になっているか
- 安全なAI利用に配慮しているか
- Markdownの構成が分かりやすいか
- 既存のルールと矛盾していないか

## Pull Requestについて

PRを送る場合は、以下を確認してください。

1. 変更理由が明確である
2. 既存の `AGENTS.md` / `PROMPT.md` の方針と矛盾しない
3. 不要な大規模変更を含めていない
4. READMEや関連ドキュメントの更新が必要な場合は反映している
